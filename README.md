 Superstore Sales & Profit Analysis Dashboard

This project presents a visual analysis of **Sales**, **Profit**, **Category performance**, and **Customer Segments** using a set of dashboards created as part of Internship Task 2
Project Overview:
The purpose of this analysis is to understand:
- Which **categories and sub-categories** generate the highest profit  
- Sales trends across time  
- Count of categories based on increase or decrease  
- Segment-wise minimum sales  
- Overall comparison between **Sales vs Profit**  

This dashboard helps business teams make data-driven decisions for improvement.

6. Min of Sales by Segment (Donut Chart)

- Home Office  
- Corporate  
- Consumer
